零次ホールドとz変換

§ステップ関数

ステップ関数\(u(t)\)を次式で定義します。

\begin{align} u(t) = \left\{ \begin{array}{@{\,}cc} 0&(t < 0)\\ 1&(t \ge 0) \end{array} \right. \label{eq:StepFunc} \end{align}

ステップ関数\(u(t)\)を使うと、 次式のように\(T\)秒間だけ\(1\)で、 それ以外は\(0\)になる関数を作れます。

\begin{align} h(t)=~u(t) - u(t-T) = \left\{ \begin{array}{@{\,}cc} 1&(0\le t < T)\\ 0&(t \le 0,~~t\ge T) \end{array} \right. \label{eq:HoldFunc} \end{align}

§零次ホールド

関数\eqref{eq:HoldFunc}を使うと時間関数\(f(t)\)を\(T\)秒間一定の値として保持(ホールド)することができます。 このように\(T\)秒間一定の値としてデータを保持することを零次ホールドといいます。 零次ホールド関数\eqref{eq:HoldFunc}を使うと、 時間関数\(f(t)~(t\ge 0)\)に対して\(T\)秒間隔でデータを採取保持(サンプルホールド)した時間関数\(\bar f(t)\)を次式で記述できます。

\begin{align} \left. \begin{array}{@{\,}rl} \bar f(t) &= f(0)h(t)\ + f(T)h(t-T) + f(2T)h(t-2T)\cdots\\ &= \displaystyle\sum_{k=0}^\infty f(kT)h(t-kT)\} \end{array} \right. \label{eq:0Hold} \end{align}

§ステップ関数と零次ホールド関数のLaplace変換

ステップ関数のLaplace変換\(U(s)\)は次式となります。

\begin{align} U(s) = \frac{1}{s} \label{eq:StepLaplace} \end{align}

したがって、ホールド関数のLaplace変換\(H(s)\)は次式となります。

\begin{align} H(s) = \frac{1-e^{-Ts}}{s} \label{eq:HoldLaplace} \end{align}

さらに\(\bar f(t)\)のLaplace変換\(\bar F(s)\)は次式となります。

\begin{align} \left. \begin{array}{@{\,}rl} \bar F(s) &= \displaystyle\sum_{k=0}^\infty f(kT)e^{-kTs}\displaystyle\frac{(1-e^{-Ts})}{s}\\ &= \displaystyle\frac{(1-e^{-Ts})}{s}\sum_{k=0}^\infty f(kT)e^{-kTs} \end{array} \right. \label{eq:FuncLaplace} \end{align}

式\eqref{eq:FuncLaplace}の\(f(kT)\)を\(f(k)\)と表記し、\(F(e^{-Ts})\)を次式で定義します。

\begin{align} F(e^{-Ts}) = \sum_{k=0}^\infty f(k)e^{-kTs} \label{eq:Fast}\\ \end{align}

式\eqref{eq:Fast}により式\eqref{eq:FuncLaplace}は次式となります。

\begin{align} \bar F(s) = \displaystyle\frac{(1-e^{-Ts})}{s}F(e^{-Ts}) \label{eq:FuncLaplace2}\\ \end{align}

§z変換

式\eqref{eq:Fast}において\(e^{-Ts}=z\)に置き換えることで\(f(k)\)のz変換を次式で定義します。

\begin{align} F(z) = \sum_{k=0}^\infty f(k)z^k = f(0) + f(1)z + f(2)z^2 + f(3) z^3+ \cdots \label{eq:zTrans} \end{align}

\(f(k)\)のz変換\eqref{eq:zTrans}の両辺に\(z\)を乗じて\(F^\prime(z)=zF(z)\)とします。

\begin{align} F^\prime(z) &=~ zF(z) = f(0)z + f(1)z^2 + f(2)z^3 + f(3) z^4+ \cdots \label{eq:zTrans2}\\ &=~ f^\prime(1)z + f^\prime(2)z^2 + f^\prime(3) z^3+ \cdots \label{eq:zTrans3} \end{align}

式\eqref{eq:zTrans2}と\eqref{eq:zTrans3}の\(z\)の係数を比較すると、\(f(k+1)=f^\prime(k)\)となっています。 つまり、\(f(k+1)\)のz変換は\(zF(z)\)になります。

逆に、\(f^\prime(k-1)=f(k)\)となっているので、\(f^\prime(k-1)\)のz変換は\(z^{-1}F^\prime(z)\)になります。
